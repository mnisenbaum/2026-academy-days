* **Metodologia:** Demonstração ---> Ação

## Abordagem Pedagógica: O Aprendizado Invertido com IA Generativa

Neste workshop, iremos explorar um novo jeito de ensinar e aprender redes. Utilizaremos o conceito de **"aprendizado invertido"** potencializado pela IA Generativa, invertendo a lógica tradicional de programação de redes do zero:

1. **Geramos** resultados rápidos com ferramentas de IA Generativa.
2. **Conhecemos** os passos lógicos e o código gerados pela IA.
3. **Aprendemos** adaptando, testando e aprimorando esses passos de forma crítica.

## Crescente de Aprendizado (Roteiros Práticos)

Nossa jornada prática envolverá a criação de topologias e a configuração de roteadores (IOS-XE) evoluindo gradativamente em complexidade de automação:

- [ ] **Lab 01:** Scripts IOS-XE gerados por IA para o CML2
- [ ] **Lab 02:** RAG com NotebookLM *(Opcional - dependente do tempo)*
- [ ] **Lab 03:** Configurar o Lab 01 usando Netmiko (script gerado por IA) enviando configs por SSH
- [ ] **Lab 04:** API Bruno: Testando solicitações RESTCONF manualmente
- [ ] **Lab 05:** Automatizando as chamadas RESTCONF
    - [ ] Script Bash (gerado por IA) 
    - [ ] Script Python (gerado por IA)
- [ ] **Lab 06:** Explorando o SDK do CML2 usando script Python (gerado por IA)
- [ ] **Lab 07:** Usando o MCP (Model Context Protocol) do CML2 para configuração via Agente de IA
