Amazon S3 (Simple Storage Service)

É o ponto de partida do RAG. Funciona como o repositório central altamente resiliente e seguro para todos os seus PDFs de topologias de rede, documentações da Cisco (manuais IOS-XE), e dumps de logs gerados no Cisco Modeling Labs (CML2).

Segurança estrita com IAM e criptografia em repouso (KMS).
Aciona notificações imediatas via S3 Event Notifications para novos arquivos.
Didática em Redes: Diga aos alunos para pensarem no S3 como a 'biblioteca física' onde todos os manuais técnicos do fabricante de rede são guardados antes de serem catalogados digitalmente.

Como programar isso? (Python Boto3)

AWS SDK for Python 
import boto3

# Inicializa o cliente do Bedrock Agent Runtime
bedrock_client = boto3.client(
    service_name='bedrock-agent-runtime',
    region_name='us-east-1'
)

def consultar_rag_aws(pergunta, kb_id):
    # Faz a busca vetorial (Retrieval) e gera resposta (Generation) usando Bedrock
    resposta = bedrock_client.retrieve_and_generate(
        input={'text': pergunta},
        retrieveAndGenerateConfiguration={
            'type': 'KNOWLEDGE_BASE',
            'knowledgeBaseConfiguration': {
                'knowledgeBaseId': kb_id,
                'modelArn': 'arn:aws:bedrock:us-east-1::foundation-model/anthropic.claude-3-5-sonnet-v1:0'
            }
        }
    )
    return resposta['output']['text']

# Pergunta técnica do aluno do Academy Days
pergunta = "Como configuro a interface do R1 para RESTCONF?"
kb_id = "AWS-KB-CML-WORKSHOP" # ID da base indexada
resultado = consultar_rag_aws(pergunta, kb_id)
print(f"Resposta Gerada pela IA:\n{resultado}")
